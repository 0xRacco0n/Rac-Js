# Rac-Js
Python tool to crawl through JS files to get endpoints and parameters and get you secrets !!

#Downloading

git clone https://github.com/0xRacco0n/Rac-Js.git 

cd cd Rac-Js/Rac-Js/

python3 rac-js.py -h


#Usage 

python3 rac-js.py -file list.txt    =======> To crawl through multiple JS files

python3 rac-js.py -url [TARGET'S URL] =======> To crawl through single JS URL

python3 rac-js.py -file list.txt -out output.txt  =======> To crawl through multiple JS files and save discovered output
